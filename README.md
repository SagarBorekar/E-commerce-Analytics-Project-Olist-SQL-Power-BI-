# E-commerce-Analytics-Project-Olist-SQL-Power-BI-
PostgreSQL + Power BI (DirectQuery) on the Olist dataset.    Performed data modeling, validation, and created optimized BI views with indexing.    Developed interactive dashboards analyzing sales, customers, logistics, payments, and sellers.    Created DAX measures (Revenue, AOV, YoY, On-time %) to generate actionable business insights.
